# NACS Monitoring & Evaluation Web Application

## Requirements
- Java JDK 17+
- Maven
- MySQL (if applicable)

## Run the Application

Clone the repository:

git clone https://github.com/HIMOLMAL/nac.git

Navigate to the project folder:

cd nac

Run the application:

mvnw.cmd spring-boot:run

Open:

http://localhost:8080
